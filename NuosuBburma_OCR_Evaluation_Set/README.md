# 规范彝文 OCR 评估集 / NuosuBburma OCR Evaluation Set

本目录是 `规范彝文 OCR / NuosuBburma OCR` 的评估集入口说明。完整评估集不直接放在 GitHub，托管在 Hugging Face 评估集仓库：

```text
https://huggingface.co/datasets/nanxidajun/NuosuBburma-OCR-Evaluation-Set
```

## 数据内容

Hugging Face 评估集当前包含复跑评估需要的最小文件：

| 文件/目录 | 说明 |
|---|---|
| `README.md` | 数据集说明 |
| `annotations.jsonl` | 最新评估标注，`758` 条 |
| `images/` | 全部被引用图片，`758` 张 |
| `samples.csv` | 每条样本的来源、场景、粒度、难度和文字构成 |
| `dataset_summary.json` | 汇总统计 |

本仓库只保留评估集入口、评估脚本和提交模型评估结果，不重复提交完整图片数据。

## 正式版本与口径

正式评估版本为 `NuosuBburma OCR Evaluation Set latest integrated 2026-06-29`。样本全部来自真实资料整理，全部纳入统一评估设置，不再拆成“展示样本”和“评分样本”两套。

当前口径：

| 组成 | 样本数 | 说明 |
|---|---:|---|
| 最新评估样本 | 758 | 最终同口径评估集 |
| 被引用图片 | 758 | 与 `annotations.jsonl` 一一对应 |
| 雪族整页子集 | 65 | 已计入最新 `758` 条总评估集 |

核心统计和结论：

| 维度 | 分布 | 结论 |
|---|---|---|
| 输入粒度 | 单行 `470` / 区域 `119` / 整页 `169` | 行图是主体，但区域和整页样本足够暴露换行、阅读顺序和页面边界问题。 |
| 场景 | 旧印刷 `507` / 新印刷 `100` / 屏幕拍照 `87` / 手写拍照 `53` / 真实照片 `11` | 评估重心放在真实旧印刷资料，同时保留拍照、屏幕和手写噪声。 |
| 难度 | easy `83` / medium `467` / hard `208` | 不是“清晰行图集合”；中高难样本构成主要压力。 |
| 文本构成 | 纯彝文 `344` / 混排 `414` | 只做粗分，避免把彝汉、注音、数字和少量特殊样本拆成过细且边界不稳定的类别。 |
| 数字 | 含数字 `177` / 不含数字 `581` | 页码、编号和工具书条目没有被过滤，数字仍单独作为弱项观察。 |

因此，这个评估集更适合衡量“真实资料整理场景下是否可用”，而不是只证明模型能否识别干净单行图。

## 构成与难度

下图汇总最终 `758` 条真实来源样本的视觉场景、输入粒度、文本构成和难度分层。逐样本明细见 `samples.csv`，质控和汇总统计见 `dataset_summary.json`。

![Evaluation dataset composition](../docs/figures/dataset_composition.svg)

难度分层不按模型输出倒推，而按来源、视觉质量、输入粒度、版面结构和文本构成情况确定：

- `easy`：清晰行图、单行、版面边界简单、干扰少。
- `medium`：彝汉混排、旧印刷、区域样本、数字/标点/注音邻近、普通场景照片。
- `hard`：整页、旧书页面、长段落、目录/脚注/页码、复杂实拍标牌和手写拍照 region/page。

## 四维度评价

| 维度 | 当前口径 |
|---|---|
| 数据规模 | `758` 条真实来源样本，`758` 张图片；line / region / page 全部计入统一评估口径。 |
| 标注准确性 | 空 GT `0`、缺图 `0`、重复 ID `0`，并提供汇总统计和质检报告。 |
| 数据多样性 | 覆盖旧印刷、新印刷、手写拍照、场景照片、屏幕拍照、整页、区域、行图、彝汉混排、拉丁注音、数字和页码。 |
| 难度合理性 | `easy 83` / `medium 467` / `hard 208`，难度由来源、视觉质量、输入粒度、版面结构和文本构成情况确定。 |

本评估集只保留真实来源样本；训练侧的合成样本和视觉增强样本不进入评估集。

## 下载

复跑评估时，可以下载到本地 `datasets/` 目录：

```bash
# 国内网络较慢时，取消下一行注释使用 Hugging Face 镜像：
# export HF_ENDPOINT=https://hf-mirror.com

hf download nanxidajun/NuosuBburma-OCR-Evaluation-Set \
  --repo-type dataset \
  --local-dir datasets/NuosuBburma_OCR_Evaluation_Set
```

`datasets/` 是本地下载目录，已在 `.gitignore` 中忽略，不作为 GitHub 仓库内容提交。

## 任务定义

输入：包含规范彝文或规范彝文混排内容的图像。

输出：图片中可见文本的 Unicode 转写，尽量保留混排关系和基本标点。

## 数据边界

- 本数据集用于评估，不是训练集。
- 样本来自真实材料，不使用合成样本作为评分材料。
- 书籍样本来源于扫描件裁剪，原始出版物版权归原出版社和权利人所有。
- 使用时请同时尊重原始材料版权与本项目的数据说明。
