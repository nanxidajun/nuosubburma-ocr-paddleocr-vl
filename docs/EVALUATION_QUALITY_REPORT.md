# 评估集标注与质量检查

## 标注方法

正式 GT 采用模型预标注、字符差异高亮和人工逐条核查。疑难规范彝文样本由熟悉规范彝文的母语者复核。GT 只记录图中可见文字，不翻译、不补全，也不根据模型输出修改图片内容。

标注位于数据集 `annotations.jsonl` 中，每条样本只有一份正式 assistant GT。

## 训练隔离

- 评估图片、GT、预测、错误类型和统计不进入训练数据。
- 训练生成器不读取评估目录、书名、样本 ID 或图像裁切。
- 形近字集合、字体、上下文、退化条件和训练权重不依据评估错例选择。
- 评估集只在模型训练和导出完成后用于报告结果。
