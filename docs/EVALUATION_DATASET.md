# 评估集说明

本文档说明评估集来源、分布、四维度评价和质控结果，对应赛事评分中的“评估集质量”。

评估集只使用真实来源样本。难度、输入粒度和文本构成都单独统计，避免只用干净行图证明模型能力。

## 设计原则

低资源文字识别可以使用合成数据训练，但不能用合成数据证明真实能力。赛事评分对评估集真实性非常敏感，因此本项目把训练数据和评估数据严格区分：

| 原则 | 说明 |
|---|---|
| 真实来源 | 评估集只使用真实来源样本，合成样本不进入评估结果 |
| 全量计入 | 最新 `758` 条样本全部进入评估集口径；没有“只展示不计分”的样本 |
| 结果口径 | 只按最终 `758` 条真实来源样本报告公开对比结果 |
| 覆盖多场景 | 覆盖新印刷、旧印刷、手写拍照、页面照片、场景/屏幕图、单行图、区域图和整页图 |
| 保留难样本 | 手写拍照、多行区域、脚注、数字、彝汉混排不因难度高而过滤 |
| 可核验 | 保留 annotations、图片、样本表和汇总统计 |

评估集保留手写、多行、旧书和混排样本。这些样本难度更高，但更接近真实输入。

## 四维度评价

| 维度 | 本评估集设计 | 可核验证据 |
|---|---|---|
| 数据规模 | `758` 条真实来源 OCR 样本和 `758` 张图片；其中 `169` 条为 page，`119` 条为 region，`470` 条为 line，能够同时评估单行识别、区域识别和页面级样本。 | Hugging Face 评估集 `annotations.jsonl`、`samples.csv`、`dataset_summary.json` |
| 标注准确性 | 空 GT `0`、缺图 `0`、重复 ID `0`，并提供质检报告。 | `docs/EVALUATION_QUALITY_REPORT.md`、Hugging Face 评估集 `dataset_summary.json` |
| 数据多样性 | 覆盖旧印刷、新印刷、手写拍照、场景照片、屏幕拍照；覆盖纯彝文、混排/注音/汉字、数字、页码、脚注、整页和多行区域。 | 本文“数据分布”“来源构成”与 `samples.csv` |

本项目把训练数据和评估数据分开：合成样本只用于训练侧补字符长尾，不进入本评估集，也不用于证明模型效果。

## 标注格式

标注使用 PaddleOCR-VL messages JSONL：

```json
{
  "id": "gen_yu_hua_line_000001",
  "images": ["images/gen_yu_hua_line_000001.png"],
  "messages": [
    {"role": "user", "content": "OCR task instruction"},
    {"role": "assistant", "content": "ground truth text"}
  ],
  "meta": {
    "source_name": "根与花",
    "source_code": "gen_yu_hua",
    "sample_type": "line",
    "script_mix": "yi",
    "difficulty": "easy"
  }
}
```

## 评估目标

| 检查项 | 目的 |
|---|---|
| 换一本书 | 判断模型是否只适应单一排版书籍 |
| 换一种版式 | 判断旧印刷、工具书、语法书、混排资料是否稳定 |
| 彝汉混排 | 检查汉语、符号、脚注和注音附近是否出现异常输出 |
| 手写拍照/照片 | 检查模型在非扫描资料上的表现 |
| 页面照片/上传图 | 检查页面图像输入下的识别稳定性 |
| 单行/多行/整页 | 检验区域识别、阅读顺序和换行边界 |

## 评价指标

主指标：

| 指标 | 含义 |
|---|---|
| 归一化编辑距离（NED） | 越低越好 |
| 空白处理后的平均编辑距离 | 忽略空白差异后的 NED |
| 规范化并忽略空白后的平均编辑距离 | Unicode 规范化和空白处理后的 NED |

辅助诊断指标：

| 指标 | 目的 |
|---|---|
| 只看彝文字符的编辑距离 | 单独观察彝文识别能力 |
| 只看汉字的编辑距离 | 单独观察汉字混排识别能力 |
| 只看数字的编辑距离 | 单独观察数字识别和格式稳定性 |
| 替换符 | 检查是否出现替换符异常输出 |
| 公式化片段 | 检查脚注/符号是否被错误公式化 |
| 拉丁字母 | 统计预测中出现拉丁字母的样本数，需要与人工标注对照 |
| 多余拉丁字母 | 检查人工标注无拉丁字母但预测多出拉丁字母的样本数 |
| 异常长输出 | 检查是否出现异常长输出 |

分组结果显示：未微调基座在最终评估集上不能作为可用模型，只作为 LoRA 微调后的对照基线。

## 真实性边界

- 评估集不进入训练包。
- 合成训练样本不进入总评估集。
- 手写拍照、区域图和整页图结果需要与单行图识别分开解读。
- 最新评估集为 `758` 条真实来源样本；未微调基座和 LoRA 微调后模型使用同一套标注、同一套图片和同一套评估脚本。
- 评估集中的书籍样本来自扫描件裁剪，原始出版物版权归原出版社和权利人所有。
