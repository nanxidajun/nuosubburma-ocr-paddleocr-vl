# 评估集说明

最终评估使用 [NuosuBburma OCR Evaluation Set](https://huggingface.co/datasets/nanxidajun/NuosuBburma-OCR-Evaluation-Set)。

## 任务

输入为包含规范彝文、汉字、少量拉丁字母、数字或标点的真实图像。模型按照视觉阅读顺序转写可见文字，保留必要行结构，不翻译、不改写，也不补全图中不存在的文字。

## 规模

| 项目 | 数量 |
|---|---:|
| 样本 / 图片 | 1,030 / 1,030 |
| 文档页 | 519 |
| OCR 实例 | 511 |
| 合成评估图片 | 0 |

评估图片全部来自真实扫描或实际拍摄，覆盖书籍扫描、手写拍照、屏幕拍照和实景拍照。内容包含纯彝文、彝汉混排，以及少量含 Latin、数字和标点的文本。

## 难度范围

| 类别 | 主要干扰 |
|---|---|
| 书籍扫描 | 旧印刷、扫描噪声、复杂版式和阅读顺序 |
| 手写拍照 | 字迹变化、透视、阴影、褶皱和纸张背景 |
| 屏幕拍照 | 摩尔纹、反光、透视、彩色背景和污渍 |
| 实景拍照 | 光照、复杂背景、尺度和拍摄角度 |

## 评分口径

最终合并模型的当前结果：

| 指标 | 结果 |
|---|---:|
| Raw Avg NED | `0.047445` |
| 去空白 Avg NED（诊断） | `0.039896` |
| NFKC + 去空白 Avg NED（诊断） | `0.034604` |

Raw Avg NED 使用原始解码字符串，不删除空白，也不做 Unicode 兼容规范化。后两项用于区分内容错误、空白差异和 Unicode 表示差异，不替代 Raw Avg NED。

## 使用边界

- 评估集只用于独立评估和错误分析，不作为训练或开发数据。
- 合成训练数据不能进入该评估集，也不能用于证明真实图片性能。
- 书籍页面的原作品权利归相应权利人所有，使用图片和标注时需遵守相关权利要求。
- 完整图片、标注与统计以 Hugging Face 数据集仓库中的当前版本为准。
