# NuosuBburma OCR 模型

完整合并模型托管在：

<https://huggingface.co/nanxidajun/NuosuBburma-OCR>

公开名称：`NuosuBburma OCR`

中文说明：规范彝文 OCR

下载：

~~~bash
hf download nanxidajun/NuosuBburma-OCR \
  --repo-type model \
  --local-dir models/NuosuBburma-OCR
~~~

Hugging Face 仓库只保留完整权重、必要配置、tokenizer、哈希清单和模型卡。训练数据、评估图片、GT、预测与内部审计文件不放入模型仓库。

完整模型卡见 Hugging Face 仓库根目录 `README.md`：<https://huggingface.co/nanxidajun/NuosuBburma-OCR>。
